Yet Another Call-graph Analysis Monitor
by Markus Sprunck
This webGL experiment visualizes the method calls of running Java applications. It can connect to a small agent (for this you have to start on your local computer). This so called YacaAgent.jar connects to VM and collects call stack information. Use up-to-date Chrome Browser for best performance. The result is a dynamic call graph.
